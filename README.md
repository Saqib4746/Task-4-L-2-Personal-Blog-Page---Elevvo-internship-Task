Personal Blog Homepage

Responsive and modern personal blog homepage built as part of Level-2, Task 4 at Elevvo Pathways.
The project focuses on clean design, accessibility, and theme customization using HTML, CSS, and JavaScript.

Overview

This project showcases a personal blog landing page with a professional, minimal layout.
It features a blue-accent (B2) theme, light/dark mode toggle, and localStorage-based theme memory for a seamless user experience.

Features

Responsive layout for desktop, tablet, and mobile

Modern blue-accent (B2) theme

Light/Dark mode toggle with localStorage persistence

Clean and accessible typography

Organized sections for About, Posts, and Contact

Smooth transitions for better user experience

Modular and maintainable file structure

Tech Stack

HTML5

CSS3

JavaScript (ES6)
